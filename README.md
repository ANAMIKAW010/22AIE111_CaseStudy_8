#Problem Description
Manual railway reservation systems often cause booking conflicts, long queues, and data management issues.
This project automates the reservation process by providing ticket booking, cancellation, PNR checking, and seat management features using Java. It reduces human error and improves efficiency.

#How to Run the Code
Install Java (JDK 8 or above).
Save the program as RailwaySystem.java.
Open terminal/command prompt.
Compile the program
